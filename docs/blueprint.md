# **App Name**: StyleSnap

## Core Features:

- Image Upload: Allow users to upload one or more primary images from their device for styling.
- Background Management: Users can select a background from a curated collection or upload their own custom image to use as a background.
- Image Composition Controls: Tools to center, scale, and drag-and-drop the uploaded foreground image onto the chosen background.
- Corner Rounding Adjustment: An interactive slider (range input) to precisely control the corner radius of the foreground image.
- Aspect Ratio Presets: Quickly apply predefined aspect ratios, including 9:16 (Reels) and 16:9 (YouTube), to the canvas.
- AI Style Suggestions Tool: An AI tool that analyzes the uploaded image to suggest optimal background choices or subtle visual adjustments for a cohesive and visually appealing final design.
- Export Final Image: Enable users to export their styled images in popular formats like PNG and JPEG.

## Style Guidelines:

- Primary action color: A vibrant, warm burnt orange (#CC7014) to signify creativity and energy without being overpowering.
- Background color: A very subtle, desaturated warm cream (#FAF7F1) to provide a clean and bright canvas that enhances the imagery.
- Accent color: A bold reddish-orange (#CC4426) to draw attention to interactive elements and highlight key information.
- Body and headline font: 'Poppins' (sans-serif) for its modern, precise, and contemporary aesthetic, suitable for clear display of short text elements.
- Utilize a consistent set of clean, minimalist line icons that clearly represent image manipulation actions (upload, scale, drag, export, crop, etc.).
- A focused, uncluttered central canvas for image editing with an easily accessible sidebar or dedicated panel for styling tools and controls, ensuring an intuitive workflow.
- Implement subtle, smooth transitions for dynamic actions such as image resizing, repositioning, and applying styling changes (e.g., corner rounding), to provide responsive user feedback.
